About This Extension

The Hattrick Match View extension provides OAuth authentication for Hattrick.org and displays enhanced match information. The extension works across all Hattrick subdomains (e.g., www85.hattrick.org, www46.hattrick.org) and includes:

  • OAuth 1.0a Authentication: Secure integration with Hattrick CHPP API
  • Connection Status Display: Visual feedback for connection state
  • Match Page Enhancement: Content script injection on match pages
  • Subdomain Support: Works on all Hattrick subdomains

Note: This is a demonstration page. To use the actual extension, load it as an unpacked extension in Chrome (chrome://extensions).
