Test Page for Overlay and Side Pane

This is a test page to verify the overlay button and side pane functionality.

When the extension is loaded and authenticated, you should see:

  • A blue "Match Data" button on the right side of the page
  • Clicking the button should open a side pane from the right
  • The side pane should display match data (or an error if not on a match page)
  • Clicking the X button in the side pane header should close it

Testing Instructions

  1. Load the extension in Chrome (chrome://extensions)
  2. Authenticate with Hattrick by clicking "Connect to Hattrick" in the extension popup
  3. Navigate to a Hattrick match page (e.g., https://www85.hattrick.org/Club/Matches/Match.aspx?matchID=76084062)
  4. Verify that the overlay button appears on the right side
  5. Click the button to open the side pane
  6. Verify that match data is loaded and displayed correctly
  7. Click the X button to close the side pane
  8. Click the button again to reopen (data should not reload)

Expected Behavior

  • Not Authenticated: No overlay button should appear
  • Authenticated + Match Page: Overlay button appears and functions correctly
  • Authenticated + Non-Match Page: No overlay button (no matchID in URL)
